Домашнее задание: bitset

Реализуйте класс bitset, представляющий битовую карту (множество целых неотрицательных чисел), размер которой может увеличиваться по мере необходимости.

Хранение данных

Битовый массив хранится в динамической памяти. Каждый бит соответствует одному числу. При установке бита с номером, превышающим текущую ёмкость, массив должен увеличиваться.

Конструкторы

· bitset() — создаёт пустое множество (ёмкость 0 или минимальная, например 64 бита)
· explicit bitset(size_t initial_capacity) — создаёт битовую карту заданного размера, все биты равны 0

Копирование и перемещение

Класс должен корректно поддерживать копирование (глубокое копирование данных), перемещение (перехват ресурсов, источник остаётся в пустом состоянии), а также копирующее и перемещающее присваивание. Деструктор освобождает выделенную память.

Основные операции

· void set(size_t k, bool b) — установить или сбросить бит k. Если k превышает текущую ёмкость, а бит устанавливается, массив расширяется: новая ёмкость должна быть не менее k+1, рекомендуется увеличивать с коэффициентом 2. Все новые биты инициализируются нулями.
· bool test(size_t k) const — проверить состояние бита k. Если k выходит за пределы текущей ёмкости, вернуть false.
· bool operator[](size_t k) const — то же самое, что test(k).

Теоретико-множественные операции

· bitset union_with(const bitset& other) const — возвращает новую битовую карту, являющуюся объединением текущей и other. Размер результата равен максимуму из размеров операндов.
· bitset intersection(const bitset& other) const — возвращает новую битовую карту, являющуюся пересечением текущей и other. Размер результата — на ваше усмотрение (минимум или максимум из размеров), обоснуйте выбор.
· bool is_subset(const bitset& other) const — возвращает true, если каждый установленный бит текущей карты также установлен в other. Если other имеет меньшую ёмкость, биты, выходящие за её пределы, считаются равными 0.

Вспомогательные методы

· size_t size() const — текущая ёмкость (количество битов, для которых выделена память)
· bool empty() const — возвращает true, если ни один бит не установлен
· void clear() — сбрасывает все биты в 0 (ёмкость не изменяется)

Требования к эффективности

· set, reset, test должны работать за O(1) (амортизированно для set с учётом реаллокации)
· union_with, intersection — за O(min(size1, size2)) или O(max) на ваше усмотрение, но с обоснованием выбора
· is_subset — за O(min(size1, size2))

Что нужно продемонстрировать

· Работу всех методов на наборах различного размера
· Корректность копирования и перемещения (отсутствие утечек, двойных удалений)
· Поведение при расширении массива
